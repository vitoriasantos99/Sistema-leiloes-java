\# Sistema de Leilões



Projeto desenvolvido em Java com integração ao banco de dados MySQL.

O sistema permite o cadastro e a consulta de produtos destinados a leilão.



\## Funcionalidades

\- Cadastro de produtos

\- Listagem de produtos cadastrados

\- Integração com banco de dados



\## Tecnologias utilizadas

\- Java

\- MySQL

\- Git

\- GitHub
